# Screenshot Context Analysis

You are analyzing a desktop screenshot cropped to focus on a specific area.

Respond with JSON describing the context:

```json
{
  "visual_description": "<2 sentences describing what you see>",
  "visible": "<pick the most appropriate category: terminal|code|messaging|meeting|browsing|reading|media|gaming|productivity>"
}
```

## Field Notes

- **visual_description**: Describe what you see in 2 good sentences for someone who is visually impaired. Describe the visual elements, layout, and content - don't transcribe text verbatim.
- **visible**: Pick the single most appropriate category for what is visible in the focus area
  - **terminal**: Command line interfaces
  - **code**: Code editors and IDEs
  - **messaging**: Chat, email, Slack, Discord, iMessage, Messages, any chat or messaging view
  - **meeting**: Video calls and conferencing such as Zoom, Webex, Meet, Teams, etc
  - **browsing**: General web browsing, social media posts, shopping
  - **reading**: Documents, articles, PDFs, documentation
  - **media**: Videos, video players, Youtube, social media images and videos
  - **gaming**: Any games or playing a game
  - **productivity**: Spreadsheets, presentations, word processors, calendars, any other workplace activities and tools

Keep responses concise and accurate.
