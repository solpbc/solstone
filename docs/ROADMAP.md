# Roadmap

Strategic direction and planned capabilities.

---

## Activities

- Global and per-facet activities management

## Data Import

- Historical bulk import support
- Automated importer framework with Plaud as first implementation

## Onboarding

- New user warmup period: collect a day of data, then suggest facets and customization for approval

## Platform Support

- Windows native observer
- iPhone and iPad apps

## Deployment

- Cloud hosting patterns with secure access (SSH tunnel initially)

## Data Management

- Integrated journal backup utilities

## Versioning

- Semver with change detection on startup from journal
- Changelog with notifications/alerts in Convey

## Social Coding

- VIT integration for agentic social coding workflows
- App creation agent (potentially store apps in journal)

## Agent Customization

- Journal-level muse directory (`<journal>/muse/`) for custom agents and frontmatter overrides

## Convey UI

- Site-wide chat bar with floating reply window over any app
  - Context-aware routing based on current view
  - Can act within app context or as general journal chat
- Rich cross-app deep linking system
