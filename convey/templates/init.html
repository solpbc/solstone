set up solstone

your private agentic partner — sol

1. set a password

protects your solstone web interface. minimum 8 characters. you can reset it anytime by using a terminal and running the command: sol password set.

this hash is stored in your journal and never transmitted

 

2. who are you?

optional — helps sol address you correctly.

 
 

3. connect gemini

optional — powers sol's thinking. gemini is the best-supported provider for full media processing. get a key at aistudio.google.com/apikey — or add other providers later in settings.

your key is stored locally in your journal config

 

need help? solstone scouts can walk you through setup.

by using gemini you agree to the google gemini API terms — your journal data is sent to google for processing

4. observers

connected observers

no observer connected yet — set up an observer to start capturing

5. get started

solstone works best with a coding agent. open a terminal, navigate to your solstone directory, and ask it to help you set up:

cd {{ repo_path }} && claude "help me set up solstone"

cd {{ repo_path }} && codex "help me set up solstone"

your data stays on your machine
